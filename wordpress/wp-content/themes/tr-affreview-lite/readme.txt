## Description

Are you a blogger, looking for beautiful free WordPress theme? Affreview should be your ready to go theme for blogging. Perfectly suitable for personal, affiliate, fashion, photography blogging. It can also be a nice theme for the business website.

* Mobile-first, Responsive Layout with the real device
* Custom Header
* Menu Description
* Google Fonts
* Wordpress Customizer
* Search Button
* The GPL v2.0 or later license. :) Use it to make something cool.
 
 ## Installation
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
5. Go to <https://www.themerally.com/product/rally-pro-multipurpose-wordpress-theme-business-blogger/>for a guide on how to customize this theme.



## Frequently Asked Questions

* The theme looks awesome, do I need to have anything extra to run it?
 
No, this theme has all blog feature and it doesn't require any third party plugin to operate. However, it is a lite version of our premium theme. You can check that out here <https://www.themerally.com/product/blog-theme-for-blogger-and-affiliate-marketers/>

Yes through wp.org and also on our website.
