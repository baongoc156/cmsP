== Description ==

Shapro is a clean and elegant WordPress theme. It is a perfect solution for any type of business, Shapro comes with a modern, sleek design and advanced functionalities.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the "shapro.zip" file. Click Install Now.
3. Click Activate to use your new theme right away.
